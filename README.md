# AppNuvem Web

Aplicação web desenvolvida utilizando PHP e MySQL para visualização e gerenciamento de dados armazenados em banco na nuvem.

Este projeto compõe o ecossistema do sistema AppNuvem, que possui versões mobile e desktop.

## Objetivo

Permitir acesso ao sistema por meio de navegador, facilitando:

- visualização de dados
- organização das informações
- integração com banco de dados remoto
- utilização sem necessidade de instalação

## Tecnologias utilizadas

- PHP
- MySQL
- HTML
- CSS
- XAMPP
- phpMyAdmin

## Funcionalidades

- listagem de dados
- cadastro de registros
- edição de informações
- exclusão de dados
- dashboard com informações organizadas
- integração com outras plataformas do sistema
